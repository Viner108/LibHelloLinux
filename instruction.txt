1.Написать header для библиотеки с объялением всех функция 
2.Написать реализацию этих функций. Не забыть добавить #include "Имя header.h"
3.Написать команду по сбору библиотеки 
g++ -shared -fPIC hello.cpp -o libhello.so
где hello.cpp - файл реализации функций, libhello.so - название созданной библиотеки, обычно название библиотеки начинается с lib а после идет само название.
4.Написать файл с использованием библиотеки. Не забыть добавить #include "Имя header.h"
5.Компилировать программу 
g++ main.cpp -L. -lhello -o hello
где main.cpp - файл, который использует библиотеку, -L - католог, где находиться библиотека, -lhello - название библиотеки без приставки lib, hello - готовый файл
6.Запукс программы 
./hello
